# Vuebly

> A Weex UI framework for WeChat, Native iOS and Android App

## Build Setup

``` bash
# install Vuebly
$ npm install -g vuebly

# create a vuebly project
$ vuebly new MyProject

# install dependencies
$ cd MyProject
$ npm install

```
